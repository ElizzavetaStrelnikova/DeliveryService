# Delivery Service System

Приложение(WebApi) для службы доставки, которое фильтрует заказы в конкретном районе города и времени обращения с и по.

## Сущности

- District. Класс, описывающий район города, в котором происходит заказ.
- Order. Класс, описывающий заказ.

## Реализация

Через API реализована возможность получить заказы, сделанные в конкретном районе в течение получаса от указанного времени.
Данные районов и заказов хранятся в json-файлах в проекте.

Изначально открывается страница со Swagger, по адресу https://localhost:44321/index.html можно через форму ввода получить результат.
